# E-Commerce Customer Analysis using Python

## Project Overview

This project analyzes customer purchasing behavior using a real-world e-commerce dataset. The goal is to clean the data, extract meaningful insights, and segment customers using RFM (Recency, Frequency, Monetary) analysis.

## Dataset

* Source: Online Retail Dataset (Kaggle)
* ~500,000 transactions
* Features include Invoice, Product, Customer, and Country details

Dataset link:
https://www.kaggle.com/datasets/vijayuv/onlineretail

Note:
The full dataset is large and not included in this repository. A smaller sample dataset is provided for demonstration.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

1. Data Loading
2. Data Cleaning (handling missing values, duplicates, returns)
3. Feature Engineering (TotalPrice, Month, Year)
4. Exploratory Data Analysis (EDA)
5. RFM Analysis (Customer segmentation)
6. Data Visualization

## Key Insights

* Most customers are low-frequency buyers
* A small group of customers contributes a major portion of revenue
* High-value customers identified using RFM analysis
* Some customers are at risk due to inactivity
